kafka-python API 모듈 개요

이 문서는 kafka-python 라이브러리의 주요 모듈을 간단한 설명과 함께 초보자도 이해하기 쉬운 형태로 정리한 것입니다.

KafkaConsumer

Kafka로부터 메시지를 읽어오는 소비자 클래스

KafkaConsumer는 Kafka 토픽으로부터 데이터를 받아오는 데 사용됩니다. 메시지를 순차적으로 읽고, 필요 시 자동으로 오프셋을 커밋하거나 파티션을 관리할 수 있습니다.

KafkaProducer

Kafka로 메시지를 보내는 생산자 클래스

KafkaProducer는 Kafka 토픽에 데이터를 전송할 때 사용됩니다. JSON이나 MsgPack 형식의 직렬화된 데이터를 보낼 수 있으며, 전송 성공/실패에 대한 콜백도 지원합니다.

KafkaAdminClient

Kafka 클러스터를 관리하는 관리자용 도구

KafkaAdminClient는 새로운 토픽을 생성하거나 삭제하고, 컨슈머 그룹이나 오프셋 등을 조회할 수 있는 고급 기능을 제공합니다. Kafka를 운영하는 환경에서 자주 사용됩니다.

KafkaClient

낮은 수준의 Kafka 브로커 연결 처리용 클라이언트

KafkaClient는 직접 사용하는 경우는 드물며, KafkaConsumer나 KafkaProducer 내부적으로 사용됩니다. 브로커와의 연결, 요청 전송 등을 처리합니다.

BrokerConnection

Kafka 브로커와의 네트워크 연결을 담당

이 클래스는 개별 Kafka 브로커와의 TCP 연결을 열고, 요청을 보내고 응답을 받는 기능을 수행합니다. 디버깅이나 커스터마이징 목적이 아니라면 직접 다룰 필요는 없습니다.

ClusterMetadata

Kafka 클러스터에 대한 메타데이터 관리

ClusterMetadata는 Kafka 클러스터에 존재하는 브로커, 토픽, 파티션 등의 정보를 저장하고 관리합니다. 프로듀서나 컨슈머가 내부적으로 사용합니다.
